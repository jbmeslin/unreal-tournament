
## Description

Node js REST API with [Nest](https://github.com/nestjs/nest)

## Installation

```bash
$ npm install
$ docker-compose up
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

Once the application is running you can visit http://localhost:3000/api to see the Swagger interface.



## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
